# Self-Assessment Period Structure Specification

## Period Flow Design

### Periods 1-5: Standard Self-Evaluation (Mid-Intern through Mid-PGY3)
1. Scholarship x
2. Wellness and Career Planning x
3. Program Feedback x
4. Assessment Review --
5. Learning 0
6. Milestone Self-Evaluation 0
7. ILP Generation 0

### Period 6: Graduation (Feb-June, PGY3)
1. Scholarship
2. Graduation Data
3. Program Feedback
4. Assessment Review
5. Board Prep
6. Milestone Self-Evaluation

### Period 7: Entering Residency (July-Sept, Intern)
1. Skills Review
2. Learning Styles and Topics
3. Goals
4. Concerns Coming Into Residency
5. Milestone Self-Evaluation

## Implementation Notes
- Period 7 is filtered by academic year (July 1 start)
- All periods include Milestone Self-Evaluation
- Career planning only in periods 1-5
- Graduation-specific content only in period 6
